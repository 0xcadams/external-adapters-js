# Chainlink External Adapter for Scrapinghub

## Input Params

- `path`: The path to query
- `result`: Optional result key to fetch (default: `avg`)

```json
{
  "jobRunID": "1",

    "min": 957378,
    "max": 1129350,
    "avg": 1043364,
    "result": 1043364
  },
  "result": 1043364,
  "statusCode": 200
}
```
